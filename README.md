# Global Development Patterns (Gapminder) — Tableau

## Overview
Interactive Tableau dashboard analyzing global development trends using GDP per capita, life expectancy, and population across 1952–2007.

## What I Built
- Animated bubble chart: GDP vs Life Expectancy (size = population, color = continent)
- Connected scatterplot showing country trajectories over time
- Country similarity map using Multidimensional Scaling (MDS)
- K-means clustering + outlier analysis (e.g., Singapore, Saudi Arabia)

## Tools
Tableau, Excel (reshaping), basic clustering + MDS outputs integrated into Tableau

## Key Insights (examples)
- High-income countries cluster together with high life expectancy and stable growth.
- Some cross-continent pairs show similar development profiles despite geography.
- Outliers reflect extreme GDP behavior or unique development paths.
